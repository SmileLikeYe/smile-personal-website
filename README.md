# Smile Personal Website

Personal site for Smile Hu, built as a product-model style portfolio.

## Stack

- React
- Vite
- Markdown content loaded from `src/content/posts/*.md`

## Commands

```bash
npm install
npm run dev
npm run build
```

## Structure

- `src/App.jsx` contains the page sections and interaction state.
- `src/styles.css` contains the responsive layout and animation system.
- `public/assets/` contains the runtime image assets used by the site.
